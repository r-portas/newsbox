# newsbox

## Tooling

- [Node.js](https://nodejs.org/)
- [pnpm](https://pnpm.io/)
- [Docker Desktop](https://www.docker.com/products/docker-desktop)

## Setup

1. Run the the Docker Compose stack
   ```bash
   docker compose up
   ```
2. Install dependencies and run the app
   ```bash
   cd app
   pnpm install
   pnpm dev
   ```
